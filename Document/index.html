How to use contact from ?

Before anything else, you need to have an SMTP configuration in place to send messages to email addresses.

What is SMTP ?

SMTP stands for "Simple Mail Transfer Protocol." It is a standard internet protocol used for sending and relaying email messages between email servers. SMTP is responsible for the process of sending outgoing emails from a client or email server to the recipient's email server, and it works collaboratively with other email protocols like POP3 (Post Office Protocol) and IMAP (Internet Message Access Protocol) that are used for receiving emails

How can i get SMTP ?

To set up an SMTP server or obtain SMTP server information, you have a few options:
Use Your Email Service Provider's SMTP: If you have an email account with a service provider like Gmail, Yahoo, Outlook, or your organization's email server, you can usually use their SMTP server information to send emails. These providers often provide SMTP server settings and authentication details.
Purchase SMTP Services: There are SMTP service providers that offer SMTP servers for sending emails. Some popular options include SendGrid, Mailgun, Amazon SES (Simple Email Service), and SMTP2GO. You can sign up for their services, and they will provide you with SMTP server details, API keys, and other necessary information.
